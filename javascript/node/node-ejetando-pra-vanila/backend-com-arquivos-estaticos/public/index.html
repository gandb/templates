Bem-vindo ao Taulukko RPG

De 2008 até 2022 o Taulukko a segunda uma plataforma de VTT para se jogar RPG, a primeira em português. Mais do que isso, foi uma comunidade. Porém com o surgimento de outras ferramentas de qualidade como o Foundry VTT, Roll D20, entre outros, decidimos parar com a ferramenta até pro conta do pouco acesso.

Porém continuamos com a vontade de colaborar com a comunidade de RPG, por este motivo continuamos com alguns projetos, eles seguem abaixo:
