/**
 * Projeto Hello World - Keycloak com Node.js, TypeScript e Docker
 * 
 * Este projeto é um exemplo básico de como integrar Keycloak com uma aplicação
 * Node.js utilizando TypeScript e containerizar a solução com Docker.
 * 
 * Objetivo:
 * Demonstrar os passos fundamentais para autenticação e autorização usando Keycloak
 * em um ambiente moderno com TypeScript e Docker.
 * 
 * Stack Tecnológico:
 * - Node.js: Runtime JavaScript
 * - TypeScript: Tipagem estática para JavaScript
 * - Keycloak: Servidor de identidade e gerenciamento de acesso
 * - Docker: Containerização da aplicação
 * 
 * Funcionalidades:
 * - Integração com servidor Keycloak
 * - Autenticação de usuários
 * - Gerenciamento de tokens JWT
 * - Proteção de rotas
 * 
 * Como usar:
 * 1. Configure as variáveis de ambiente para Keycloak
 * 2. Execute `docker-compose up` para iniciar os containers
 * 3. Faça requisições para as rotas autenticadas com o token JWT
 */
